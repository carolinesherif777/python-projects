# 🐍 Snake Game

A classic Snake Game developed in Python using the Turtle graphics library. This project was built as part of my Python learning journey to strengthen my understanding of programming fundamentals through a hands-on game.

## ✨ Features

* Smooth snake movement using keyboard controls
* Random food generation
* Snake grows after eating food
* Real-time score tracking
* High score tracking during gameplay
* Wall collision detection
* Self-collision detection
* Automatic game reset after collision

## 🛠️ Technologies Used

* Python
* Turtle
* Random
* Time

## 📚 Concepts Practiced

* Variables and data types
* Conditional statements
* Loops
* Functions
* Lists
* Event handling
* Collision detection
* Object movement
* Game loop design
